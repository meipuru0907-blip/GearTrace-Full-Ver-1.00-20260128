# GearTrace アプリケーション説明書

**GearTrace** は、機材（Gear）の資産管理、持ち出しリストの作成、コスト管理を効率化するためのウェブアプリケーションです。

## 主な機能一覧

### 1. ダッシュボード (Dashboard)
アプリケーションのホーム画面です。全ての機材状況を一目で確認できます。
- **資産概要**: 総資産価値、月間/年間のランニングコストを表示します。
- **検索・フィルター**:
    - キーワード検索（モデル名、メーカー、カテゴリー）
    - カテゴリー別フィルター
    - ステータス別フィルター（全て、稼働中、故障など）
- **表示モード切り替え**: グリッド表示、リスト表示、詳細グリッド表示から選択可能です。
- **財務分析**: 詳細な資産価値の分布やサブスクリプションのコスト内訳をモーダルで確認できます。

### 2. 機材管理 (Gear Management)
機材の詳細な情報を登録・管理します。
- **機材登録**: メーカー、モデル名、カテゴリー、購入日、価格、シリアル番号などを登録できます。
- **ステータス管理**: 「Available（稼働中）」「Maintenance（メンテナンス中）」などの状態を管理できます。
- **コンテナ機能**: バッグやケースなど、他の機材を収納する「コンテナ」として機材を定義できます。
- **数量管理**: 機材の在庫数量を管理します。
- **耐用年数**: 資産としての耐用年数を設定できます（デフォルト5年）。

### 3. パッキングリスト (Packing Lists)
撮影や旅行などのイベントごとに持ち出しリストを作成できます。
- **リスト作成**: 日付と名前を指定して新しいリストを作成します。
- **機材追加**: 在庫から機材を選択してリストに追加します（在庫数の上限チェックあり）。
- **コンテナ連携**: コンテナを選択すると、その中身（Contents）を一括でリストに追加する機能があります。
- **PDF出力**: 作成したリストをPDF形式で出力・印刷できます。
- **ラベル印刷**: コンテナなどのラベルを印刷する機能があります。

### 4. サブスクリプション管理 (Subscriptions)
機材に関連する定額制サービスや保険などのコストを管理します。
- **コスト可視化**: 支払いサイクル（月払い/年払い）と次回支払日を管理し、ランニングコストを算出します。

### 5. 設定・データ管理 (Settings)
データのバックアップや外部連携用の機能です。
- **Excel出力**:
    - **資産台帳**: 資産管理用のフォーマットで全機材データを出力します。
    - **確定申告用データ**: 税務処理に便利な形式でデータを出力します。
- **JSONバックアップ/復元**: 全データをJSONファイルとしてバックアップ保存し、必要な時に復元できます。
- **便利ツール**: 耐用年数ガイド（Excel）のダウンロードが可能です。
- **サポート**: バグ報告や機能要望を送るフォームへのリンクがあります。

## データ保存について
本アプリケーションはブラウザのローカルデータベース（IndexedDB）を使用しており、データはお使いのブラウザ内に保存されます。重要なデータは定期的に「設定」画面からJSONバックアップを取得することをお勧めします。
